Aadhar Card online web application

It consists of Two section:

Admin Portal:

In this application admin can do certain operation like,

  • 1. Login through admin credentials
  • 2. Approve new Aadhaar Card request
  • 3. Verify request for duplicate Aadhaar
  • 4. Display all issued Aadhaar Card
  • 5. Delete Aadhaar card details for dead citizen

    • User Portal:

    In this application user can do certain operation like,

  • 1. Sign in to apply for a new Aadhar Card
  • 2. Login to see the Aadhar number assigned by the admin
  • 3. Update address, phone number, and date of birth of Aadhaar Card
  • 4. Request duplicate Aadhaar Card

    • Project information

    • ToolKit: Eclipse, Visual Studio Code
    • Frontend: HTML, CSS, Angular, Typescript
    • Backend: JAVA
    • Database: MySQL
